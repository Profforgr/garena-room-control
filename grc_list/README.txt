grc_list generates a list of rooms based on roomen.sqlite.

Usage
-----

Copy roomen.dat from your Garena installation to grc_list/bin/roomen.sqlite. Then, run grc_list.jar. A rooms.txt text file will be outputted with a list of rooms. Note that roomen.sqlite must be in the directory from which you run grc_list.jar. Also remember that file names are case sensitive on Unix-based operating systems.
